# 🍽️ Spice Garden Restaurant Website

## 📌 Project Overview

Spice Garden Restaurant Website is a modern, responsive, and user-friendly website developed for a local restaurant business. The website helps customers explore the restaurant's menu, view food galleries, read customer reviews, and easily contact the restaurant for reservations and inquiries.

This project was developed as part of **Future Interns – Full Stack Web Development Task 3 (2026): Build, Pitch & Monetize a Real Local Business Website**.

---

## 🎯 Project Objective

The goal of this project is to create a professional online presence for a local restaurant that helps:

* Increase customer trust
* Improve business visibility
* Showcase menu offerings
* Provide easy contact options
* Enhance customer engagement
